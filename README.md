# azec
Azure Rest API wrapper that makes Azure operation easier

## Installation
```
$ pip install azec
```

## Usage
Listing all Azure App Service
```
$ azec webapp ls
```

Listing all Azure Functions
```
$ azec func ls
```

Listing all Azure VM
```
$ azec vm ls
```

Listing all Azure SQL Server
```
$ azec sqlsv ls
```

Listing all Azure SQL Database
```
$ azec sqldb ls
```
